# Simple Blog with i18n

## Features
- JavaScript Standard Style (http://standardjs.com)
- ES7 async functions
- express
- sequelize
- validation
- template pug (rename jade)
- csrf
- i18n
- and more

## Future
- auth
- upload images
- captcha
- pagination
- socket.io

## Installation
- set config in .env
```bash
$ npm i
$ bower i
$ npm start or npm run dev
```
